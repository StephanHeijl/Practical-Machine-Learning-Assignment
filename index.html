Practical Machine Learning Course Submission

Stephan Heijl, 14/05/2015

This is my submission for the Practical Machine Learning Course. I approached this project in the following fashion:

  1. Exploratory Data visualization using Weka
  2. Low variation data exclusion
  3. Principal Component Analysis
  4. Testing different types of machine learning algorithms
  5. Optimization of Random Trees algorithm
  6. Prediction result analysis

Exploratory Data visualization using Weka

The PML training set consists of over a hundred of different variables. Using the open source machine learning program Weka[1], I was able to quickly visualize the different ranges of data.

Low variation data exclusion

A large array of variables can be shown not contain significant variation in their data. This results in data that is either useless for machine learning, since its presence does not indicate a certain class, or can even lead to overfitting. Using R's NearZeroVar function allowed me to filter these attributes, resulting in a leaner, more relevant dataset. This dataset was quicker to train and less likely to overfit on irrelevant data.
